# Typingerz JS Runner

## 使い方
1. Chrome の拡張機能管理画面を開く
2. 「デベロッパーモード」を有効にする
3. このフォルダを「パッケージ化されていない拡張機能を読み込む」で指定する
4. 拡張機能アイコンをクリックして ON/OFF を切り替える
5. 「詳細」または「オプション」を開いて、実行したい JavaScript を保存する

## 挙動
- ON: `typingerz.com` とサブドメインで保存済み JavaScript を自動実行します
- OFF: 注入を止めます。保存済みコードは残ります
- 見た目: アイコンの色と `ON / OFF` バッジで状態が分かります

## 注意
- 同じページ内遷移で再実行されることがあります。
- 入力欄が欲しい場合は options.html の textarea に直接貼り付けて保存してください。
